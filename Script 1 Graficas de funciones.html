Gráficas de funciones en python¶

Para graficar funciones en python primero tener definida la funcin y se debe construir una tabla de valores de tal manera que lo que se grafican son los pares ordenados $(x, f(x))$

Ejemplo: graficar la funcin $f(x) = \sqrt{x + 4}$ en el intervalo $[-4, 8]$.

El cdigo sera algo como este:

In [35]:
import numpy as np
import matplotlib.pyplot as plt 

x = np.array([-4, -2, 0, 2, 4, 6, 8])
y = np.sqrt(x + 4)
plt.plot(x,y)

[<matplotlib.lines.Line2D at 0x7e715c0>]

Explicación del código:¶

En las dos primeras lneas se importan las librerias de python que contienen los mtodos para construir vectores y grficas. En particular:

  1. El mdulo Numpy https://www.numpy.org : es una librera de python muy usada para computacin cientfica y lgebra matricial.

  2. El mdulo Matplotlib https://matplotlib.org : es una libreria de python muy til para crear grficos.

La sentencia " np.array " se usa para crear un vector de nmeros reales.

La sentencia " np.sqrt " indica que se esta usando la funcin raz cuadrada.

Observe que en ambas sentencias se antepone "np" indicando que las funciones se encuentran dentro de la librera Numpy la cual la estamos llamando con el alias np (primera lnea del script).
